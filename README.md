# Rdio

Inspired by the
[Rdio alfred workflow](http://dferg.us/rdio-workflow-for-alfred-2/), and modeled
like the built-in iTunes actions with the following actions:

* Play
* Play/Pause (toggle)
* Stop
* Next/Previous Song
* Add to Collection

Planned features:

* Search Rdio

## Installation

Download ZIP and run the file!

## Requirements

* The Rdio app
